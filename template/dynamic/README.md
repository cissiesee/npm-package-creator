## 使用指南
### 安装
    npm install <%=packageName %> --save-dev
### 使用
    import { } from "<%=packageName %>";
## 开发指南
### 安装依赖包
    npm install
### 开发
    npm run dev
### 编译
    npm run es5
### 打包
    npm run build
### 测试
    npm run test
### 持续测试
    npm run test:watch
